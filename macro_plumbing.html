📖 Effet Cantillon & liquidité nette

📖 Cycle de crédit de Minsky

📖 Criticalité de Bak (gamma des dealers)

GEX > 0 : les dealers vendent les hausses / achètent les baisses → marché auto-stabilisant, range-bound. GEX < 0 : ils amplifient → trending, breakouts violents. Le flip positif→négatif = changement de régime, réduire la taille. VVIX < 80 = complaisance (tout le monde short vol), > 120 = panique.

📖 Hiérarchie temporelle

Les forces macro sont classées du plus lent au plus rapide : crédit (trimestres) → liquidité (semaines-mois) → microstructure (jours-semaines) → emploi avancé (mois) → divergences (semaines). On ne trade pas contre les forces lentes : si crédit + liquidité sont verts, on peut être long même si les rapides clignotent ; mais si le crédit se retourne, rien ne sauve le marché à moyen terme.
