§99 Robustness Lab — DSR + MinTRL (López de Prado / Bailey 2014)

DSR (Deflated Sharpe Ratio) corrige le Sharpe observé pour le biais de sélection multiple : après n_trials stratégies testées, le max attendu par hasard peut être élevé — DSR mesure la probabilité que le Sharpe soit supérieur à ce maximum. MinTRL (Minimum Track Record Length) indique combien d'années de données sont nécessaires pour prouver statistiquement que le Sharpe dépasse un benchmark SR=0.5. Source : lab/robustness_metrics.json généré 2026-05-16T12:45:56.608171 (29 setups). Affichage : top 25 par DSR DESC.
Méthodologie : Bailey & López de Prado (2014). DSR = Φ((SR − E[max SR_N]) / σ(SR)). MinTRL years = 1 + (1 − γ·SR + ((κ−1)/4)·SR²) · (Φ⁻¹(0.95) / (SR − 0.5))². Source : lab/robustness_metrics.json.
