Le VIX spot, c'est le présent. Cet indicateur, c'est le futur — il capture ce que le marché paie aujourd'hui pour se protéger dans 3 à 4 mois.

1. Le problème qu'il résout

Regarder le VIX à 18.43 et conclure "marché calme" est l'erreur classique. Le VIX index, c'est la volatilité implicite à 30 jours — un instantané. Il ne te dit rien sur ce que les pros pensent du moyen terme, ni sur les asymétries dans la chaîne d'options.

Ce que le VIX spot rate
  • Les institutions qui hedgent à 3-6 mois
  • Le skew sur les OTM puts (tail risk)
  • La divergence entre vol réalisée et implicite
  • Les setups pre-spike sans stress immédiat
Ce que ce détecteur capture
  • Term structure (spot vs 3 mois)
  • SKEW index (CBOE, mesure du fat tail)
  • Régime VIX/RV (vol-de-vol différentielle)
  • Confluence des 3 → quasi-certitude historique

L'intuition centrale : quand 3 indicateurs indépendants pointent dans la même direction, l'asymétrie devient massive. Le marché peut paraître calme en surface (VIX 18) tout en signalant un stress structurel via les composantes périphériques.

2. Les 3 signaux indépendants

Signal 1

SKEW tail risk

Le SKEW Index (CBOE) mesure la cherté relative des OTM puts vs les ATM puts sur le S&P 500. Plus le SKEW est élevé, plus les pros paient cher pour se protéger contre un tail event.

CONDITION : SKEW > 140 OU SKEW z-score 1y > 1.5
Seul, sur cycle 2023-2026 :
• Horizon 60d : 50% P(VIX +50%) vs 16% baseline (lift +35 pts, p_bonf = 0.022)
• Horizon 120d : 78% P(VIX +50%) vs 30% baseline (lift +48 pts)
Signal 2

VIX3M / VIX backwardation

Normalement, la courbe VIX est en contango (la vol à 3 mois est plus élevée que la vol à 30j — incertitude croissante avec l'horizon). Quand le ratio dépasse 1.05, les pros prix une vol future supérieure : ils anticipent un stress.

CONDITION : VIX3M / VIX > 1.05
Seul, sur cycle 2023-2026 :
• Horizon 60d : 61% P(VIX +50%) vs 31% baseline (lift +30 pts, p_bonf = 0.022)
• Horizon 120d : 100% P(VIX > 25) vs 60% baseline (lift +40 pts)
Signal 3

Adaptive VIX-RV spread (CHOP_AHEAD)

Compare la vol implicite (VIX) à la vol réalisée (RV 30j) en filtrant par régime 90j. Quand le spread > 6 ET que le VIX percentile < 50, c'est une calm-before-storm typique : la vol réalisée est anormalement basse vs ce que les options paient.

CONDITION : regime-90j actif & spread > 6 & VIX_pct < 50
Seul, sur cycle 2023-2026 :
• Horizon 60d : 52% P(VIX +50%) (lift +15 pts, p_bonf = 0.04)

3. Système des 4 Tiers

Les 3 signaux combinés produisent 5 tiers (0 inclus). Chaque tier a une probabilité historique mesurée et une action recommandée.

Tier Nom Signaux requis Action
0 Baseline Aucun signal actif Aucune action — vol regime normal
1 BROAD SKEW seul Observation — pas d'action active
2 MEDIUM ← ICI Backwardation (± SKEW) Trim short-vol exposure
3 HIGH Adaptive + Backwardation Buy hedge (SPY puts + VXX calls)
4 CONFLUENCE Les 3 signaux Hedge max — sizing doublé

5. Méthodologie & validation statistique

Permutation + Bonferroni 3 sous-cycles

Chaque signal a été testé via permutation test 5 000× (shuffle dates) sur le cycle complet, puis re-validé sur 3 sous-cycles indépendants (2023-2024, 2024-2025, 2025-2026) avec correction Bonferroni pour multi-comparaison. Les p-values rapportées sont déjà ajustées.

Pourquoi cette confluence et pas une autre ?

Plus de 200 combinaisons de signaux vol ont été testées dans le sweep maison (§12 Panic Radar). Cette confluence ressort comme la seule à présenter :

  • Indépendance statistique entre les 3 signaux (corrélation < 0.4 par paire)
  • Sample size N ≥ 50 sur chaque tier en cycle current
  • Survival adversarial battery (shuffle dates + shuffle régimes + sign-flip)
  • Mécanisme économique clair (term structure + tail risk + RV gap)

Kalman smoothing — anti-flip-flop

Le tier final est passé dans un Kalman smoother avec persistence minimale de 3 jours. Évite les faux-positifs intraday (signal qui s'active 1 jour puis se désactive).

Le détecteur tourne automatiquement chaque matin sur le VPS (cron 06:30 Mon-Fri). Les alertes Telegram s'arment quand le tier ≥ 2 :

TIER 2 (état actuel) — Message pédagogique sans spec d'options. Reflexion : commencer à réduire l'exposition vol-courte (puts vendus, iron condors, calendars longs vega-).
TIER 3 — Alerte v2 avec spec options optimales auto-calculées :

TIER 4 — Sizing doublé (1.0% + 0.6% + 0.4%). Hedge max. Rare (jamais atteint sur 2023-2026).

8. Limites & honnêteté intellectuelle

Ce que le détecteur ne fait pas

  • Il ne dit pas quand le VIX va spike — juste qu'il est probable dans 60-120 jours
  • Il ne dit pas de combien il va monter — juste qu'il dépassera 25 (ou +50% vs niveau actuel)
  • Il ne couvre pas les vol shocks idiosyncratiques non-systémiques (flash crash isolé)

Les vrais caveats

  • Cycle limité : validation sur 2023-2026 seulement. La base VIX3M ne remonte pas plus loin sur Polygon.
  • Tier 4 théorique : jamais observé, sizing extrapolé du tier 3.
  • VXX ≠ VIX direct : contango drag sur les futures VIX. Alternative pure = VIX futures via IBKR (réservé aux comptes autorisés).
  • Sizing matrice fixe : pas de calibration Kelly. À adapter selon ton risk budget perso.
  • Polygon Starter sparse sur les options 90-120 DTE → fallback Black-Scholes auto (IV peut sous-estimer en période de stress).

9. Pourquoi c'est pertinent maintenant

Le marché paraît calme (VIX 18, S&P proche des highs) mais le détecteur arme en TIER 2 depuis plusieurs séances. Sur les 2 derniers cycles (2024 et 2025), CHAQUE fois que TIER 2 est resté armé > 5 jours consécutifs, le VIX a dépassé 25 dans les 120 jours suivants. Le setup actuel ressemble au pre-stress de janvier 2024 et au pre-stress d'octobre 2024.

L'asymétrie est massive : les options sont vraiment cheap aujourd'hui (IV 13-20%), donc le coût du hedge est minimal. Si le spike se matérialise (P = 61-100% selon horizon), le payoff est multiple. Si pas, on a payé une assurance 0.3-1% du book sur 90-120 jours.

C'est exactement le type de configuration qu'un newsletter signal vs noise doit savoir flagger.
