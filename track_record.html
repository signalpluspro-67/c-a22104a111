Methodology. Equity curves are close-to-close, equal weight. The real book marks the open published positions daily (cash when none are open). The philosopher profiles run through the same basket engine over signal_archive (score ≥ 65), rebalanced biweekly, top 10. All KPIs (Sharpe, max drawdown, alpha/beta, up/down capture, information ratio, R squared) come from one shared function so every profile is measured the same way, against SPY. Data: Polygon EOD through 2026-06-22. Paper backtests are not a promise of future returns, and a six-month real sample is too small to call an edge. As always, this is what happened, not advice.
